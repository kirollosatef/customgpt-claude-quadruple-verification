All done. Here's the complete Next.js 14 project structure:

**Root config files (7):**
- `package.json` — Next.js 14, React 18, TypeScript, Tailwind, ESLint dependencies
- `tsconfig.json` — Strict TypeScript with `@/*` path alias pointing to `./src/*`
- `next.config.mjs` — Next.js configuration
- `tailwind.config.ts` — Tailwind CSS with content paths for `src/`
- `postcss.config.mjs` — PostCSS with Tailwind and Autoprefixer plugins
- `.eslintrc.json` — ESLint extending `next/core-web-vitals`
- `next-env.d.ts` — Next.js TypeScript declarations
- `.gitignore` — Standard Next.js ignores

**App Router files (5):**
- `src/app/layout.tsx` — Root layout with Inter font, nav bar (Home/About links), and `<main>` wrapper
- `src/app/globals.css` — Tailwind directives + light/dark CSS variables
- `src/app/page.tsx` — Home page with welcome content
- `src/app/about/page.tsx` — About page
- `src/app/api/health/route.ts` — GET endpoint returning `{ status: "ok", timestamp: "..." }`
